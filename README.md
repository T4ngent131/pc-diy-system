# PC DIY 智能装机系统

> 一套面向用户装机选配、订单提交、库存管理与测试验证的前后端分离项目。

## 功能模块

### 用户端

- **首页**：品牌介绍、热门方案推荐、使用流程展示。
- **注册登录**：支持用户注册、登录、登录态保存和退出登录。
- **自定义装机**：支持 CPU、显卡、主板、内存、硬盘、电源、机箱、散热器等配件选择。
- **兼容性校验**：实时检测 CPU/主板插槽、内存类型、电源功率、机箱尺寸等兼容性问题。
- **推荐配置**：提供入门、主流、高端、旗舰 4 套推荐方案。
- **提交订单**：支持根据推荐方案或自定义配置生成订单，并扣减库存。
- **订单查询**：查看订单状态和配置摘要。
- **选购指南**：提供 CPU、显卡、内存、电源、散热等知识库文章。

### 管理与数据

- **配件管理**：按类型查询配件，支持品牌筛选和低库存查询。
- **库存管理**：支持入库、出库、库存不足校验和库存变更记录。
- **订单管理**：支持订单列表查询、订单状态更新和配置明细查看。
- **基础数据**：包含 240 条配件数据、8 篇知识库文章和 4 套推荐配置。
